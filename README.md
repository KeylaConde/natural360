# Natural360 - Your Eco-Friendly Online Store
Natural360 is an e-commerce platform focused on promoting sustainable and eco-friendly products.

## About the Project

Natural360 is an e-commerce platform designed to promote suatainable and ecofriendly products. This project is built using **HTML, CSS, JavaScript, and Local Storage** to offer a seamless and user-friendly shopping experience.

## Features

✅ Clean and responsive design  
✅ Product catalog with images, descriptions, and prices  
✅ Add to cart functionality using Local Storage  
✅ Simple checkout process  
✅ Fully customizable for future enhancements  

## Tech Stack

- **HTML** - Structuring the website
- **CSS** - Styling and responsiveness
- **JavaScript** - Interactivity and dynamic behavior
- **Local Storage** - Storing cart data

## Future Improvements

- Integration with a payment gateway
- User authentication system
- Database implementation for better product management
